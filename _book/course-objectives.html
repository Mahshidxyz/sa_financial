A short course on Survival Analysis in the Financial Industry

Chapter 1 Course objectives

The software used in the course is the statistical language R and the IDE (Integrated Development Environment) RStudio. A basic prior knowledge of both is assumed.

The required packages for the course are:

# Install packages
install.packages(c("survival", "condSURV", "JM"))
devtools::install_github("noramvillanueva/clustcurv")

The codes in the notes may assume that the packages have been loaded, so it is better to do it now:

# Load packages
library(survival)
library(condSURV)
library(JM)
library(clustcurv)
